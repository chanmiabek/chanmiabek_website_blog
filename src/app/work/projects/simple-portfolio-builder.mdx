## Overview

This portfolio brings Chan Miabek's profile, projects, technical skills, and contact links into a focused web experience. The goal was to make the site easy to scan, simple to maintain, and ready for future project updates.

## Key Features

- **Responsive layout**: Designed to work across desktop and mobile screens.
- **Structured content**: Profile, work, gallery, and project data are managed from reusable content files.
- **Project pages**: Each project has its own detail page with summary, images, and implementation notes.
- **Contact-ready profile**: Social links and email access are visible from the about page and footer.

## Technologies Used

- **Next.js** for the application framework.
- **React** for component-based UI.
- **TypeScript** for safer development.
- **MDX** for editable project content.

## Outcome

The site gives Chan a clear online presence and a foundation for publishing new projects, learning notes, and professional updates.
