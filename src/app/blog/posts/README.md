# How to Add a Blog Post

Create a new `.mdx` file in this folder.

Example:

```mdx
---
title: "My New Post"
subtitle: "Optional short subtitle."
publishedAt: "2026-07-01"
summary: "A short summary of the post."
image: "/images/coding.jpg"
tag: "Learning"
team:
  - name: "Chan Miabek"
    role: "Software Developer"
    avatar: "/images/chan-profile.jpg"
    linkedIn: "https://www.linkedin.com/chanmiabek/"
---

Write your blog post here.
```

The file name becomes the URL. For example, `my-new-post.mdx` becomes `/blog/my-new-post`.
